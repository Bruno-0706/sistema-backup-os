# sistema-backup-os
Um projeto da disciplina de SO, explorando o conceito de Sistemas de Arquivos por meio de um simples sistema de backups.

``` C
findLatestBackupTime()
```
Retorna o tempo, em `time_t` do último arquivo de backup.

---

```C
createBackup()
```
Uma função resposável por criar o arquivo de backup no seu respectivo diretório de backup
Retorna `1` se deu certo ou `0` se deu errado.
